接码平台API文档

通用说明

所有API支持GET和POST请求方法，认证令牌可通过以下方式提供：

  1. URL参数: ?token=您的令牌
  2. POST请求体JSON: {"token": "您的令牌"}
  3. 请求头: Authorization: Bearer 您的令牌（优先级最低）

状态码说明

状态码 描述
200 请求成功
201 创建成功
400 请求参数错误
401 认证失败
403 权限不足
404 资源不存在
429 请求过于频繁
500 服务器内部错误

注意事项

  • 所有API都有频率限制，请勿频繁请求
  • 取号后需要在15分钟内获取验证码，否则号码将自动释放
  • 所有时间默认使用UTC时间，特别注明的除外
  • 余额不足时，取号将失败
  • API密钥应妥善保管，请勿泄露
